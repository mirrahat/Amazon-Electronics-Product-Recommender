Overview

This project is a personalized product recommendation system built with collaborative filtering using matrix factorization (SVD). It predicts likely user interests and handles cold-start users with fallback logic. Built using Python, pandas, Surprise, and evaluated with RMSE and Precision

Dataset

Dataset: ratings_Electronics.csv which consists of almost one million user's data from Amazon's open data repository. Contains user_id, product_id, rating, and timestamp.

  • Dropped timestamp and duplicates.
  • Removed missing values.
  • Performed EDA to explore user/product activity and rating distribution.
  • Prepared with Surprise’s Reader and Dataset for training.

Model Information

Model: Singular Value Decomposition (SVD) from the Surprise library.

  • Trained on an 80/20 train-test split.
  • Evaluated using Root Mean Squared Error (RMSE).
  • Benchmarked against KNN, NMF, and Baseline models.
  • Top-N product recommender based on predicted ratings.
  • Although the Baseline model achieved a lower RMSE than SVD, SVD was chosen due to its ability to capture latent user-item interactions.
  • For large datasets (e.g., 1M+ records), simple models may overfit to frequent interactions, whereas SVD effectively generalizes by decomposing the user-item matrix.
  • SVD reduces noise in sparse recommendation data and improves personalization, making it more suitable for real-world recommendation systems despite slightly higher RMSE.

Limitations

  • Cold-start problem for new users/products.
  • Requires retraining for new data.
  • Implicit preferences only, lacks contextual data.
