Overview

This system is a personalized product recommendation engine built using collaborative filtering (SVD). It intelligently suggests products to users based on their historical preferences and behavior, or offers fallback recommendations to new users.

Dataset

The system uses a large-scale e-commerce dataset containing user ID, product ID, rating, and timestamp. The data was preprocessed and used to train a Surprise SVD model for collaborative filtering.

Model

The model uses Singular Value Decomposition (SVD) from the Surprise library to predict unseen ratings. It was trained on 100% of the dataset using build_full_trainset(), and recommendations are based on the top-N predictions. A shuffle option allows randomized presentation for demo purposes.

About

Created by Mir Hasibul Hasan Rahat, a software developer and data science enthusiast. This project showcases skills in backend development, machine learning, and user-centered design using Flask.

Limitations

  • No real product metadata (name, image, etc.) in the dataset
  • Cold-start problem partially solved using popular/hidden gem fallback
  • Model retraining required for updated predictions

Future Improvements

  • Include product names and images using external metadata
  • Add user session tracking and login features
  • Support content-based and hybrid recommendation models
